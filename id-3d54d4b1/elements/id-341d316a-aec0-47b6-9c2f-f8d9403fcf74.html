Definition: Interoperability Requirement ABB is a Requirement being formulated as an agreed normative statement in functional terms on a legal, organisational, semantic, or technical attribute of a To-Be European Public Service Source: European Commission, ISA Programme Source reference: (https://ec.europa.eu/isa2/sites/default/files/eif_brochure_final.pdf) Example: The following implementation is an example on how this specific Architecture Building Block (ABB) can be instantiated as a Solution Building Block (SBB): Ewshopp: - Interoperability requirements are for facilitating access to and integration of data within the project.
