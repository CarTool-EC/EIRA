Data Syntax ()
Definition: Data Syntax ABB is a Data Object defining the way in which data is put together with appropriate identifiers, delimiters, separator character(s), and other non-data characters to form messages. Source: ISO Source reference: https://www.iso.org/obp/ui/#iso:std:iso:14817:-1:ed-1:v1:en:term:4.59 Example: The following implementation is an example on how this specific Architecture Building Block (ABB) can be instantiated as a Solution Building Block (SBB): XML syntax - XML syntax refers to the rules that determine how an XML application can be written. The XML syntax is very straight forward, and this makes XML very easy to learn.
