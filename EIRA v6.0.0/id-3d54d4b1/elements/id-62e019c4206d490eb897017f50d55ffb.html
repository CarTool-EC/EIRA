Definition: GraphQL ABB is a Technology Service that implements the behaviour of a server-side runtime that execute queries using a type system you define for your data Source: GraphQL Source reference: https://graphql.org/learn/ Example: The following implementation is an example on how this specific Architecture Building Block (ABB) can be instantiated as a Solution Building Block (SBB): Github is using GraphQL Server and API to allow users retreive the required data without having additional information that is not relevant for its purpose. https://docs.github.com/es/graphql LOST view: Technical view - Infrastructure Interoperability saliency: GraphQL Server ABB contributes to the semantic and technical interoperability because it set up the API that permits (i) access to the public administration information using semantics, (ii) exposes API following up common specifications related to GraphQL language.
