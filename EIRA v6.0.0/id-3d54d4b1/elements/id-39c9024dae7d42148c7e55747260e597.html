Telemetry ()
Definition: Telemetry ABB is a Technology Service that enables the procedures to automatically measure and analyse information about public administration services and IT platforms to track performance and identify issues. Source: EEA Source reference: https://www.eea.europa.eu/help/glossary/gemet-environmental-thesaurus/telemetry Example: The following implementation is an example on how this specific Architecture Building Block (ABB) can be instantiated as a Solution Building Block (SBB): Telemetry is a clear example of use in computers integrated in spacecrafts. https://www.esa.int/Enabling_Support/Space_Engineering_Technology/Onboard_Computers_and_Data_Handling/Telemetry_Telecommand LOST view: Technical view - Infrastructure Interoperability saliency: Telemetry is salient for semantic and technical interoperability because it sets up the mechanisms to monitor a specific ICT system to (i) collect behavioural information, and (ii) understand the ICT system's behaviour during its execution.
