EIF Interoperability Level ()
Definition: EIF Interoperability Level ABB is a Business Object setting guidelines for developing public services. The EIF defines four interoperability levels, which cover legal, organisational, semantic and technical interoperability. Each level deserves special attention when a new European public service is established. Source: European Commission, ISA Programme. The New European Interoperability Framework (EIF) for European public services. [Online] 2017 Source reference: (https://ec.europa.eu/isa2/eif_en.)
